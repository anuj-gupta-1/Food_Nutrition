Food Nutrition Database API

Welcome to the Food Nutrition Database API. This service provides access to nutritional information for various food products.

Available Endpoints

GET /products.csv

Download the complete product database in CSV format.

Usage: https://food-nutririon.web.app/products.csv

Content-Type: text/csv

Database Information

  • Total Products: 11,302+
  • Categories: Dairy, Snacks, Beverages, and more
  • Data Sources: StarQuik, JioMart, and other retailers
  • Last Updated: October 2025

CSV Format

The CSV file contains the following columns:

  • id, product_name, brand, category, subcategory
  • size_value, size_unit, price, source, source_url
  • ingredients, nutrition_data, image_url
  • energy_kcal_per_100g, carbs_g_per_100g, protein_g_per_100g, fat_g_per_100g
  • last_updated, search_count, data_quality_score
